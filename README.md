## About Task Manager

Task Manager is an app for managing personal tasks

## Tools Used

- Laravel 9
- Tailwind
- VueJs
- MySQL
- Sail

## Setting up the project

    1.Install docker https://www.docker.com/
    2.Run local dev environment with `sail up`
    3.Launch Project http://localhost
